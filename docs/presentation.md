## 슬라이드 2: 문제 정의 1 - 도서관 좌석 부족

### 제목
도서관의 한계: 좌석 부족 문제

### 핵심 콘텐츠

#### 왼쪽 섹션: 수치 데이터
```
📊 서울여대 도서관 현황

학생 수              약 9,000명
도서관 면적          약 10,446㎡
도서관 총 좌석       1,622석

1인당 좌석 비율
  현재: 0.18석/인
  권장: 0.3-0.4석/인
  
결과: 권장 기준 대비 45-60% 부족
```

#### 오른쪽 섹션: 공강 시간 현황
```
⏰ 공강 시간대 피크 타임

점심 시간
12:00 - 15:00

이 시간대에
도서관 이용 수요 급증
↓
자리 찾기 위해 돌아다님
↓
시간 낭비
```

### 디자인 요소
- 왼쪽에 큰 수치 강조 (0.18 vs 0.3-0.4)
- 오른쪽에 시간표 형태로 배치
- 빨간색으로 "부족" 강조
- 간단한 아이콘으로 시각화

### 스피커 노트
"첫 번째 문제는 도서관 좌석 부족입니다. 서울여대는 약 9,000명의 학생에 1,622개의 좌석만 있습니다. 이는 학생 1명당 0.18개의 좌석을 의미하는데, 교육부 권장 기준인 0.3-0.4석/인과 비교하면 45-60% 부족합니다. 특히 점심시간인 12시부터 3시 사이에 공강이 집중되면서 도서관이 매우 혼잡해집니다."

## 슬라이드 3: 문제 정의 2 - 미활용 자원

### 제목
미활용 자원: 빈 강의실의 현실

### 핵심 콘텐츠

#### 상단: 문제 구조
```
왼쪽 (빨간색)              오른쪽 (파란색)
도서관은 포화 상태 ←→ 강의실은 비어있음

1,622석이 부족     약 175개의 강의실
모두 차있음        대부분 비어있음
```

#### 중앙: 강의실 현황 정보
```
🏢 강의실 현황

5개 건물
├─ 50주년 기념관: ~59개
├─ 인문사회관: ~46개
├─ 조형예술관: ~24개
├─ 제1과학관: ~22개
└─ 제2과학관: ~24개

총 약 175개의 강의실
```

#### 하단: 현황 분석
```
❌ 현재 문제점
• 강의실이 비어있어도 학생들이 알 수 없음
• 도서관 좌석 부족 문제와 강의실 미활용이 공존
• 학교 자원이 효율적으로 활용되지 않음
```

### 디자인 요소
- 대비 구도: 도서관(혼잡) vs 강의실(비어있음)
- 아이콘 사용으로 시각적 강조
- 건물 아이콘과 숫자 조합
- 문제점은 빨간색 X 마크로 표시

### 스피커 노트
"두 번째 문제는 미활용 자원입니다. 학교에는 5개 건물에 약 175개의 강의실이 있습니다. 이 강의실들 중 많은 수가 특정 시간에 비어있습니다. 하지만 학생들은 언제 어느 강의실이 비어있는지 알 수 없습니다. 결과적으로 도서관은 포화되고 강의실은 비어있는 악순환이 벌어지고 있습니다."

## 슬라이드 4: 문제 정의 3 - 기존 시스템의 문제점

### 제목
기존 시스템: 너무 복잡하고 불편하다

### 핵심 콘텐츠

#### 상단: 프로세스 흐름
```
학교 종합정보시스템을 통한 공강교실 검색

1️⃣ 시스템 접속
   ↓
2️⃣ 로그인 (인증 필요)
   ↓
3️⃣ 메뉴 탐색
   ↓
4️⃣ 학부/학년/학과 등 상세 조건 입력
   ↓
5️⃣ 추가 필터 선택
   ↓
6️⃣ 시간표 조회
   ↓
7️⃣ 결과 확인
   ↓
8️⃣ 강의실 상황 파악

총 최소 8번 이상의 클릭 필요!
```

#### 하단: 문제점 정리
```
⏱️ 공강 시간이 시작되는 바로 그 순간...

❌ 복잡성: 8회 이상 클릭 필요
❌ 로그인: 매번 인증 필요
❌ 상세 조건: 학부, 학년, 학과 등 복잡
❌ 속도: 시간이 오래 걸림
❌ 직관성: 결과 확인이 어려움

💡 결과: 공강 시간을 낭비하며 강의실을 찾음
```

### 디자인 요소
- 프로세스 플로우를 화살표로 표현
- 숫자 원형 아이콘(1️⃣ ~ 8️⃣) 사용
- 문제점은 빨간색 X 마크
- 타이머 아이콘으로 시간 낭비 강조

### 스피커 노트
"세 번째 문제는 기존 시스템의 복잡함입니다. 현재 학교 종합정보시스템에서 공강교실을 찾으려면 최소 8번 이상 클릭해야 합니다. 로그인도 필수이고, 학부, 학년, 학과 같은 복잡한 조건들을 입력해야 합니다. 공강이 생기는 바로 그 순간에 빨리 확인하고 싶은데, 이런 과정 때문에 귀중한 공강 시간을 낭비하게 되는 것입니다."

## 슬라이드 5: 해결 방안 1 - 빈강의실 서비스 소개

### 제목
빈강의실: 3단계로 빈 강의실을 찾다

### 핵심 콘텐츠

#### 상단: 핵심 메시지
```
🎯 간단한 3단계 검색으로 빈 강의실을 즉시 제공!
```

#### 중앙: 3단계 프로세스 (시각적 강조)

```
Step 1️⃣               Step 2️⃣               Step 3️⃣
건물 선택        →     요일 선택        →     시간 선택
─────────            ─────────             ─────────

• 50주년 기념관   • 월              • 1-2교시
• 제1과학관       • 화              • 3-4교시
• 제2과학관       • 수              • 5-6교시
• 인문사회관      • 목
• 조형예술관      • 금


                    ✅ 완료!
                    ↓
            빈 강의실 결과 표시
            └─ 강의실명
            └─ 층 정보
            └─ 시설 정보
```

#### 하단: 주요 특징
```
💡 빈강의실의 특징

✓ 간결성: 단 3개 선택지로 완료
✓ 속도: 검색 결과를 즉시 확인
✓ 정확성: 학교 공식 시간표 기반
✓ 접근성: 로그인 없이 누구나 사용 가능
```

### 디자인 요소
- 3단계 프로세스를 화살표로 연결
- 각 단계를 다른 색상으로 강조
- Step 아이콘(1️⃣ 2️⃣ 3️⃣) 사용
- 선택지는 버튼 형태로 표현

### 스피커 노트
"이제 저희의 해결 방안을 소개합니다. 빈강의실은 단 3단계로 빈 강의실을 찾을 수 있습니다. 첫째, 원하는 건물을 선택합니다. 50주년 기념관, 제1과학관, 제2과학관, 인문사회관, 조형예술관 중 하나를 고르면 됩니다. 둘째, 요일을 선택합니다. 월요일부터 금요일 중 원하는 날을 고릅니다. 셋째, 시간을 선택합니다. 1-2교시, 3-4교시, 5-6교시 중 하나를 선택하면, 바로 그 시간에 비어있는 강의실이 표시됩니다."

## 슬라이드 6: 해결 방안 2 - 기존 시스템과의 비교

### 제목
빈강의실 vs 기존 시스템: 확실한 차이

### 핵심 콘텐츠

#### 비교 표 (왼쪽: 기존 시스템 | 오른쪽: 빈강의실)

```
┌─────────────────┬────────────────────────┬──────────────────────┐
│     항목        │   기존 시스템          │    빈강의실          │
├─────────────────┼────────────────────────┼──────────────────────┤
│ 로그인          │ ✓ 필수                 │ ✗ 불필요            │
├─────────────────┼────────────────────────┼──────────────────────┤
│ 클릭 횟수       │ 최소 8회 이상 ❌       │ 3회 ✅               │
├─────────────────┼────────────────────────┼──────────────────────┤
│ 조건 입력       │ 복잡함                 │ 간단함               │
│                 │ (학부, 학년, 학과 등)  │ (건물, 요일, 시간)   │
├─────────────────┼────────────────────────┼──────────────────────┤
│ 검색 속도       │ 느림 (시스템 로드)     │ 즉각적 ⚡            │
├─────────────────┼────────────────────────┼──────────────────────┤
│ 사용 시기       │ 사전 예약 필요         │ 지금 바로 확인 ⏰     │
├─────────────────┼────────────────────────┼──────────────────────┤
│ 접근성          │ 학교 계정 필요         │ 누구나 접속 가능     │
├─────────────────┼────────────────────────┼──────────────────────┤
│ 직관성          │ 낮음 (복잡한 UI)       │ 높음 (3가지만 선택)  │
└─────────────────┴────────────────────────┴──────────────────────┘
```

#### 핵심 강조 (하단)
```
🔑 가장 중요한 차이점

기존 시스템
로그인 후 최소 8번 이상의 클릭과 
복잡한 조건 입력 필요

빈강의실 ⭐⭐⭐
지금 바로 3번의 클릭
(건물 → 요일 → 시간)
만으로 빈 강의실을 찾을 수 있음!
```

### 디자인 요소
- 표 형식으로 명확하게 비교
- 기존 시스템은 검은색/회색, 빈강의실은 파란색/초록색으로 구분
- ✓ 와 ✗ 마크로 명확하게 표시
- 숫자 강조 (8회 vs 3회)

### 스피커 노트
"이 표에서 보시는 것처럼, 기존 시스템과 빈강의실은 완전히 다릅니다. 기존 시스템은 로그인을 해야 하고, 최소 8번 이상 클릭해야 하며, 복잡한 조건을 입력해야 합니다. 반면 빈강의실은 로그인이 필요 없고, 단 3번의 클릭만으로 완료되며, 간단합니다. 무엇보다 기존 시스템은 미리 준비해야 하지만, 빈강의실은 공강이 생기는 바로 그 순간에 바로 확인할 수 있습니다."

## 슬라이드 7: 해결 방안 3 - 서비스의 가치

### 제목
빈강의실이 제공하는 가치

### 핵심 콘텐츠

#### 4가지 가치 제안 (각각 아이콘 포함)

```
1️⃣ 공강 시간 효율적 활용
   ┗─ 공부할 공간이 있어 공강 시간을 낭비하지 않음
   ┗─ 도서관이 가득 찼을 때의 대안 제공

2️⃣ 팀 작업 공간 확보
   ┗─ 조별 과제를 위한 모임 장소 확보
   ┗─ 그룹 스터디 공간 확보
   ┗─ 프로젝트 진행 장소 확보

3️⃣ 정보 접근의 용이성
   ┗─ 복잡한 절차 없이 즉시 정보 제공
   ┗─ 누구나 쉽게 사용 가능
   ┗─ 스트레스 감소

4️⃣ 학교 자원 활용 극대화
   ┗─ 비어있는 강의실이 제대로 활용됨
   ┗─ 학교 시설의 효율성 증대
   ┗─ 자원 낭비 감소
```

#### 하단: 학생들의 경험 개선
```
📈 기대 효과

Before (기존)            →    After (빈강의실)
─────────────              ────────────────
공강이 생김                공강이 생김
  ↓                           ↓
도서관 가봄              바로 빈강의실 확인
  ↓                           ↓
자리 없음                 편한 강의실 선택
  ↓                           ↓
길 떠남                   해당 강의실 이동
  ↓                           ↓
시간 낭비 ❌               효율적 활용 ✅
```

### 디자인 요소
- 4가지 가치를 카드 또는 박스로 표현
- 각 가치마다 관련 아이콘 추가 (⏰ 📚 💡 🏢 등)
- Before/After 비교 플로우 사용
- 화살표와 체크마크로 개선 시각화

### 스피커 노트
"빈강의실이 제공하는 가치는 네 가지입니다. 첫째, 학생들은 공강 시간을 효율적으로 활용할 수 있습니다. 도서관이 가득 찼을 때 강의실이 대안이 되어, 공강 시간을 낭비하지 않습니다. 둘째, 팀 작업 공간을 확보할 수 있습니다. 조별 과제나 그룹 스터디를 위한 장소를 쉽게 찾을 수 있습니다. 셋째, 정보 접근이 매우 용이합니다. 복잡한 절차 없이 즉시 정보를 얻을 수 있어 스트레스를 줄일 수 있습니다. 넷째, 학교 자원이 더 효율적으로 활용됩니다. 비어있는 강의실이 제대로 활용되어 학교 시설의 효율성이 극대화됩니다."

## 슬라이드 8: 서비스 시연

### 제목
실제 서비스 시연

### 핵심 콘텐츠

#### 상단: 시연 방식 안내
```
🖥️ 실시간 웹사이트 시연

다음과 같은 순서로 시연하겠습니다:
```

#### 중앙: 시연 순서
```
Step 1️⃣: 빈강의실 메인 페이지 접속
         └─ 간단하고 직관적인 인터페이스 소개

Step 2️⃣: 건물 선택 (예: 제2과학관)
         └─ 5개 건물 중 선택하는 과정

Step 3️⃣: 요일 선택 (예: 월요일)
         └─ 월~금 중 선택하는 과정

Step 4️⃣: 시간 선택 (예: 1-2교시)
         └─ 3개 시간대 중 선택하는 과정

Step 5️⃣: 검색 결과 확인
         └─ 빈 강의실 목록 표시
         └─ 강의실명, 층, 시설 정보 표시

Step 6️⃣: 결과 해석
         └─ 학생이 선택한 시간에 비어있는 강의실 확인
         └─ 편리하게 강의실 선택
```

#### 하단: 핵심 기능 요약
```
💡 핵심 기능 3가지

✓ 간단한 선택 인터페이스
  → 건물, 요일, 시간 3개만 선택하면 완료

✓ 즉각적인 결과 표시
  → 검색 결과가 바로 화면에 나타남

✓ 상세한 정보 제공
  → 강의실명, 층, 시설 정보까지 제공
```

### 디자인 요소
- Step별로 화면 레이아웃 다이어그램 표시
- 실제 웹사이트 스크린샷 또는 목업 이미지 삽입 가능
- 화살표와 숫자로 순서 표시
- 최종 결과 화면 강조

### 스피커 노트
"이제 실제 서비스를 시연해 드리겠습니다. 웹사이트에 접속하면 깔끔하고 직관적인 인터페이스가 보입니다. 먼저 원하는 건물을 선택합니다. 예를 들어 제2과학관을 선택하겠습니다. 다음으로 요일을 선택합니다. 월요일을 선택하겠습니다. 그리고 시간을 선택합니다. 1-2교시를 선택합니다. 이제 검색 버튼을 클릭하면, 바로 제2과학관의 월요일 1-2교시에 비어있는 강의실 목록이 표시됩니다. 각 강의실에는 이름, 층, 그리고 어떤 시설이 있는지까지 자세히 보여줍니다. 학생들은 이 정보를 보고 자신의 필요에 맞는 강의실을 선택할 수 있습니다."
